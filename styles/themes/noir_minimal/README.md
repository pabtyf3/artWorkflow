# Noir Minimal Theme

**noir_minimal** is a high-contrast, restrained style theme intended for scenes that emphasize
clear silhouettes, limited palettes, and sharp lighting separation.

This theme is designed to:
- prioritize value contrast over texture
- keep palettes limited and controlled
- encourage crisp, readable forms
- remain flexible for individual artistic interpretation

It does **not** enforce a specific composition or lighting setup.

---

## Intended Use

This theme works well for:
- detective or noir environments
- graphic novel-style scenes
- moody interiors with strong shadows
- minimalist game environments

---

## What This Theme Provides

### Tool Availability
- A restrained brush set focused on clean edges
- Limited blending modes to preserve contrast
- Simple erasers for non-destructive correction

### Colour Guidance
- Limited palette with low saturation
- High value contrast
- Emphasis on separation between light and shadow

### Texture Guidance
- Minimal, subtle grain
- Textures should never dominate the value structure

### Workflow Hints
- Clear separation between depth layers
- Lighting kept on separate layers for controlled edits

---

## What This Theme Does NOT Do

- It does not choose colours for you
- It does not apply brushes automatically
- It does not paint or generate art
- It does not decide lighting or composition

The artist remains in full control.
